To run the code, please run the following commands:
	javac NGramRunner.java
	java NGramRunner
	--OR--
	java NGramRunner @CHAR1 @CHAR2
	
where @CHAR1 = any first character
and   @CHAR2 = any second character (only used for trigrams)

Running without parameters allows for command-line entering of characters for text generation.

The computing environment used was 64-bit Java version 1.8.0_181.
